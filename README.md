# Email Cleaner

Mark all emails from a specific sender as read.

## Setup environment

* 3.10
* requirements.txt

### First time
```bash
conda create -n email_cleaner python=3.10 -y
conda activate email_cleaner
pip install -r requirements.txt
```

### Next time
```bash
conda activate email_cleaner
```
